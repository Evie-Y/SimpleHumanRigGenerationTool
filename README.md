# SimpleHumanRigGenerationTool
The goal of this tool is to help make a simple rig generation tool. This will help cut the repetition of rigging and speed up the basic controls on a humanoid character.

### Features
- I will include different control shapes based on slection order of the joints.
- The control’s size will be based on parameters selected.

### Goals
- Create an easy-to-use UI window and improve on PySide2.
- Make sure the tool works on different kinds of joints.
- Create unique controls.
- Help speed up making a functioning rig.
- Improve on GitHub commits and naming.
- Have controls and groups follow to common joint naming.

### Inputs
- Selected Joints
- Control Radius
- FK or IK rig
- Control shapes

#### UI
- Buttons for creating an fk or ik.
- Checkbox for circle controls or unique controls.
- Dropdown with button to create unique curves, not rigged.
